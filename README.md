# Cloud Services Dashboard

A Cloud Services Management Dashboard built with **Next.js**, **TypeScript**, and modern frontend architecture.

## Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Zustand
- React Query
- Recharts

## Features

- Authentication (Mock)
- Real-time metrics dashboard
- Server management
- Resource monitoring
- Dark / Light theme

## Getting Started

```bash
npm install
npm run dev
```
